# Как проверить ERP на мобильных и планшетах

## 🚀 Быстрый способ (DevTools браузера)

### Шаг 1: Откройте приложение
```
http://localhost:3000
```

### Шаг 2: Откройте DevTools
- **Chrome/Edge:** Нажмите `F12` или `Ctrl+Shift+I`
- **Firefox:** Нажмите `F12` или `Ctrl+Shift+I`

### Шаг 3: Включите режим устройства
- Нажмите `Ctrl+Shift+M` (Toggle Device Toolbar)
- Или нажмите на иконку устройства в панели инструментов

### Шаг 4: Выберите устройство

#### 📱 Мобильные телефоны:
- **iPhone SE** (375x667) - маленький телефон
- **iPhone 12/13** (390x844) - стандартный телефон
- **Samsung Galaxy S20** (360x800) - Android телефон

#### 📱 Планшеты:
- **iPad** (768x1024) - стандартный планшет
- **iPad Pro** (1024x1366) - большой планшет
- **Surface Pro** (912x1368) - Windows планшет

#### 💻 Десктоп:
- **Desktop** (1920x1080) - стандартный монитор

## 📱 Проверка на реальных устройствах

### Вариант 1: Через локальную сеть

1. **Узнайте IP адрес компьютера:**
   ```bash
   # Windows
   ipconfig
   # Найдите "IPv4 Address", например: 192.168.1.100
   
   # Mac/Linux
   ifconfig
   # Найдите "inet", например: 192.168.1.100
   ```

2. **Убедитесь, что сервер запущен:**
   ```bash
   npm run dev
   ```

3. **На мобильном устройстве:**
   - Подключитесь к той же Wi-Fi сети
   - Откройте браузер
   - Введите: `http://192.168.1.100:3000`
   - Замените `192.168.1.100` на ваш IP адрес

### Вариант 2: Через ngrok (для тестирования из интернета)

1. **Установите ngrok:**
   - Скачайте с https://ngrok.com/
   - Или через npm: `npm install -g ngrok`

2. **Запустите ngrok:**
   ```bash
   ngrok http 3000
   ```

3. **Используйте полученный URL:**
   - Скопируйте URL вида: `https://xxxx-xx-xx-xx-xx.ngrok.io`
   - Откройте на любом устройстве (даже не в той же сети)

## 📋 Что проверить на мобильных/планшетах

### ✅ Базовые проверки:

1. **Sidebar (Боковое меню)**
   - [ ] Открывается по кнопке меню
   - [ ] Закрывается при клике на overlay
   - [ ] Закрывается при выборе пункта меню
   - [ ] Плавная анимация открытия/закрытия

2. **Header (Верхняя панель)**
   - [ ] Кнопка меню видна на мобильных
   - [ ] Заголовок не обрезается
   - [ ] Кнопка сохранения адаптивная

3. **Контент**
   - [ ] Карточки отображаются корректно
   - [ ] Таблицы прокручиваются горизонтально
   - [ ] Формы удобны для заполнения
   - [ ] Кнопки достаточно большие для касания (44px минимум)

4. **Навигация**
   - [ ] Все разделы доступны
   - [ ] Переключение между разделами работает
   - [ ] Поиск работает корректно

### 📱 Специфичные проверки для мобильных:

- [ ] Текст читаемый (не слишком мелкий)
- [ ] Поля ввода не увеличиваются при фокусе (iOS)
- [ ] Скролл плавный
- [ ] Нет горизонтального скролла (кроме таблиц)
- [ ] Модальные окна открываются корректно
- [ ] Toast-уведомления видны

### 📱 Специфичные проверки для планшетов:

- [ ] Используется пространство эффективно
- [ ] Карточки в 2-3 колонки
- [ ] Таблицы полной ширины
- [ ] Графики масштабируются корректно

## 🔧 Устранение проблем

### Проблема: Не могу открыть на мобильном
**Решение:**
1. Проверьте, что сервер запущен
2. Убедитесь, что устройства в одной сети
3. Проверьте firewall (может блокировать порт 3000)
4. Используйте ngrok для внешнего доступа

### Проблема: Сайт не адаптивный
**Решение:**
1. Очистите кэш браузера
2. Проверьте, что используется правильный viewport meta tag
3. Перезапустите сервер разработки

### Проблема: Таблицы не прокручиваются
**Решение:**
- Это нормально для мобильных устройств
- Таблицы должны иметь горизонтальный скролл
- Проверьте класс `overflow-x-auto` на таблицах

## 📝 Заметки

- **Минимальный размер touch target:** 44x44px (iOS стандарт)
- **Рекомендуемый размер текста:** минимум 16px для мобильных
- **Breakpoints:** sm:640px, md:768px, lg:1024px, xl:1280px
