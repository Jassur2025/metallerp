## Разрешения для тестирования

### Мобильные устройства
- iPhone SE (375x667)
- iPhone 12/13 (390x844)
- Samsung Galaxy S20 (360x800)
- Pixel 5 (393x851)

### Планшеты
- iPad (768x1024)
- iPad Pro (1024x1366)
- Android Tablet (800x1280)

### Десктоп
- Small (1024x768)
- Medium (1280x720)
- Large (1920x1080)

## Общие проверки

- [ ] Все модальные окна открываются корректно
- [ ] Все формы можно заполнить на мобильных
- [ ] Все кнопки кликабельны (минимум 44x44px)
- [ ] Текст читаем (минимум 14px)
- [ ] Нет горизонтального скролла на мобильных (кроме таблиц)
- [ ] Навигация доступна на всех устройствах
- [ ] Toast-уведомления видны на всех экранах

## Инструменты для тестирования

1. **Chrome DevTools**
   - F12 → Toggle device toolbar
   - Выбрать устройство из списка

2. **Responsive Design Mode**
   - Firefox: Ctrl+Shift+M
   - Chrome: F12 → Device toolbar

3. **Онлайн инструменты**
   - BrowserStack
   - Responsively App
   - LambdaTest

## Рекомендации по исправлению

### Проблема: Горизонтальный скролл
**Решение:** Добавить `overflow-x-auto` к контейнеру таблицы

### Проблема: Маленькие кнопки
**Решение:** Использовать `min-h-[44px] min-w-[44px]` для кнопок

### Проблема: Нечитаемый текст
**Решение:** Использовать `text-sm` или больше для мобильных

### Проблема: Модальные окна выходят за экран
**Решение:** Добавить `max-h-screen overflow-y-auto` к модальному окну
